Chief Companion — quick install reference
=========================================

What this program does:
  • Reads iRacing telemetry locally on this PC
  • Streams session data to chiefracing.com for AI coaching
  • Saves your iRacing setups + SimuCube/SimMagic profiles automatically
  • Speaks coaching in your ear while you drive

What it does NOT do:
  • No game files modified
  • No background updates pushed without your consent
  • No data sent unless you sign in and pair via your install token

Where things live:
  • Program:        %ProgramFiles%\Chief Racing\Chief Companion\
  • Per-user data:  %APPDATA%\Chief\config.json   (your token + cached settings)
  • Saved setups:   %APPDATA%\Chief\setups_archive\
  • Hardware:       %APPDATA%\Chief\hardware_archive\

How to use:
  1. Open chiefracing.com → sign in → Dashboard → Install Companion
  2. Copy your install token
  3. Run Chief Companion → paste token → click Activate
  4. A green flag icon appears in your Windows tray (bottom-right)
  5. Launch iRacing → drive → Chief talks to you in your ear

To uninstall:
  • Settings → Apps → "Chief Companion" → Uninstall
  • Or:  Start Menu → Chief Racing → Uninstall Chief Companion

Help / diagnostics:
